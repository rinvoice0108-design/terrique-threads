# 수건 전문 지식 — 포스트 작성 참고 백과

## 소재별 특성

### 이집트면 (Egyptian Cotton)
- 섬유 길이 38mm 이상 — 세계 면화 생산량의 1% 미만 희귀 원료
- 나일강 미네랄 토양이 섬유 품질 결정
- 실크처럼 부드럽고, 세탁할수록 더 부드러워지는 특성
- 흡수력과 내구성 모두 최고급

### 터키면 (Turkish / Aegean Cotton)
- 에게해 연안 재배, 섬유 길이 길고 흡수력 뛰어남
- 테리 직물(Terry cloth)의 원조 — "테리"라는 이름이 터키에서 유래
- 흡수 빠르고 건조도 빠름 — 실용성과 품질 균형 탁월

### 코마사 (Combed Cotton)
- 일반 면에서 짧은 섬유와 불순물을 빗질로 제거한 고급 원사
- 표면이 매끄럽고 보풀 적음, 내구성 향상
- 호텔 수건에 주로 사용하는 이유

### 링 방적사 (Ring Spun Cotton)
- 섬유를 촘촘히 꼬아 만든 원사 — 균일하고 강도 높음
- 반복 세탁에 강해 업소용 수건에 적합

### 제로 트위스트 (Zero Twist)
- 꼬임 없이 만든 원사 → 루프가 더 부드럽고 흡수 빠름
- 단점: 반복 세탁 시 내구성이 일반 코마사보다 약함

### 오가닉 코튼 (Organic Cotton)
- 농약·화학비료 없이 재배 (GOTS 인증)
- 아기·아토피·민감 피부에 적합
- 일반 면보다 환경 부담 약 46% 감소

### 극세사 (Microfiber)
- 흡수력 면의 5~7배, 건조 속도 3배 빠름
- 단점: 세탁 시 미세플라스틱 방출, 뜨거운 물 세탁 불가, 피부 마찰 강함

### 대나무 (Bamboo)
- 자연 항균, 통기성 좋음, 독특한 부드러움
- 실제 대나무 섬유 비율이 낮은 제품이 많음 — 원료 비율 표기 확인 필요
- 린넨보다 부드럽고 면보다 건조 빠름

### 린넨 (Linen)
- 여름에 특히 좋음 — 면보다 2~3배 빠른 건조
- 빳빳한 질감으로 피부 마사지 효과
- 세탁할수록 부드러워지는 특성

## GSM 심화 지식
- GSM = Grams per Square Meter (1㎡당 무게)
- 무게가 높을수록 루프가 촘촘하고 흡수력 ↑, 건조 속도 ↓
- 300GSM: 여름·스포츠·여행·캠핑용 (빠른 건조)
- 400~450GSM: 일반 가정용 세면 (균형형)
- 500~550GSM: 고급 가정용, 선물용
- 600GSM 이상: 호텔급 (최고 흡수력, 건조 느림)
- 같은 GSM이라도 루프 밀도 차이로 실제 품질 달라짐

## 수건 관리 핵심 지식

### 첫 세탁의 중요성
- 새 수건에는 방적유(sizing) 코팅 → 처음엔 물이 튕겨나감
- 세제 없이 또는 소량 세제로 2~3회 세탁 후 본래 흡수력 발휘
- 첫 세탁 시 찬물로 색 고정 (색빠짐 방지)

### 섬유유연제의 진실
- 유연제 성분(quaternary ammonium)이 루프 표면을 코팅 → 흡수력 최대 50% 감소
- 단기적으로 부드럽지만 장기적으로 수건을 망침
- 대체법: 식초 반 컵 (냄새 제거 + 섬유 연화) / 베이킹소다 (세균 제거 + 냄새 중화)

### 세탁 온도 가이드
- 30℃: 색빠짐 방지, 에너지 절약 (일반 컬러 수건)
- 40~60℃: 일반 위생 세탁 (대부분 면 수건)
- 60℃ 이상: 병원·업소용 살균 세탁
- 90℃ 삶기: 완고한 냄새 최후 수단 (흰 수건만)

### 건조가 가장 중요한 이유
- 세탁 후 30분 내 꺼내기 — 방치 시 세균 증식, 냄새 원인
- 건조기: 루프가 부풀어 오히려 더 부드러워짐 (저온 권장)
- 자연건조: 직사광선 (살균효과), 반드시 통풍 확보
- 반건조 상태에서 접지 말 것 — 냄새의 핵심 원인

### 냄새의 과학
- 원인: 세균(Micrococcus) 번식 → 지방산 분해 → 불쾌한 냄새 발생
- 해결: 60℃ 세탁 or 햇빛 건조 or 베이킹소다 + 식초 병행
- 식초는 세탁기 헹굼 단계에 넣을 것 (세제와 동시 사용 금지)

### 수명과 교체 시기
- 일반적으로 1~2년 (매일 사용 기준)
- 교체 신호 4가지: 냄새 안 빠짐 / 흡수력 뚝 떨어짐 / 보풀 심함 / 섬유 얇아짐
- 관리 잘 하면 3년 이상 사용 가능

## 흔한 오해 교정 (팁형 포스트에 활용)
- "유연제 쓰면 더 좋아진다" → 단기 부드러움, 장기 흡수력 파괴
- "두껍다고 좋은 수건이다" → GSM 낮으면 루프가 비어 흡수력 낮음
- "흰 수건은 관리 어렵다" → 온도 세탁·표백 가능해서 오히려 위생 관리 쉬움
- "수건은 다 비슷하다" → 원사·GSM·직조 방식에 따라 수명 2~3배 차이
- "냄새는 세탁으로 없앤다" → 건조가 세탁보다 중요

## 용도별 선택 가이드

### 일반 가정 세면용
- GSM 450~500, 코마사 면 권장
- 세면 타월(34×85cm) + 목욕 타월(70×140cm) 구분 사용

### 선물·답례품용
- GSM 500~600, 코마사 또는 이집트면
- 흰색·아이보리: 고급스러움 + 세탁 용이
- 포인트 컬러: 행사 테마 맞춤

### 업소용 (펜션·게스트하우스)
- 링 방적사, GSM 400~450 (빠른 건조·반복 세탁 내구성)
- 흰색 권장: 고온 세탁·표백 가능

### 아기·민감 피부용
- 오가닉 코튼 or 대나무 혼방
- 무형광·무방부제·OEKO-TEX 인증 필수
- GSM 300~400 (가볍고 빠른 건조)

## 수건과 피부의 관계
- 얼굴 전용 수건을 따로 쓰면 피부 트러블 감소 (피부과 권장)
- 마구 문지르지 않기 — 가볍게 두드려 수분 흡수
- 머리 수건 터번은 10분 이내 — 장시간은 두피 열 손상
- 수건 2~3일에 1회 교체 권장 (세균 번식 억제)

## 자수·맞춤 제작 지식
- 자수 vs 프린팅: 자수가 내구성 높고 고급스러움, 프린팅은 세밀한 디자인 가능
- 자수 실 소재: 폴리에스터(내구성) vs 레이온(광택)
- 자수 위치: 모서리·중앙 자수는 흡수 방해 최소화
- 소량 제작: 최소 10~30장부터 가능한 업체 많음
- 납기: 자수 제작은 발주 후 2~4주 소요 일반적

## 인증 종류 정리
- OEKO-TEX Standard 100: 유해물질 100가지 이상 검사 (테리크 보유)
- GOTS (Global Organic Textile Standard): 오가닉 원료 + 제조 공정 전체 인증
- BCI (Better Cotton Initiative): 지속가능한 면화 재배 기준
- FAIR TRADE: 생산자 공정 임금 보장 인증
