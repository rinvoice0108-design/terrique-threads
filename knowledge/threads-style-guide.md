# Threads 잘하는 브랜드 스타일 가이드

테리크와 카테고리가 유사한 라이프스타일/생활용품 브랜드 중
Threads에서 반응이 좋은 패턴을 정리한 레퍼런스입니다.

---

## 브랜드별 핵심 패턴

### 올리브영 (생활용품 · 라이프스타일)
- **첫 줄**: 계절/날씨/오늘 상황으로 즉시 공감 유도
- **구조**: 공감 → 문제 → 해결(제품) → 부드러운 권유
- **말투**: 친한 언니/동생처럼. "~하지 않으세요?", "사실 저도요"
- **길이**: 6~10줄, 2줄씩 단락 나누기
- **특징**: 정보 먼저, 브랜드는 자연스럽게 뒤에

예시 구조:
```
요즘 이런 고민 있지 않으세요?
(공감 포인트 1줄)

사실 원인은 생각보다 단순해요.
(핵심 정보 2~3줄)

이렇게 해보세요.
(실용 팁)

#해시태그
```

---

### Lush (뷰티 · 윤리적 소비)
- **첫 줄**: 가치/철학을 질문으로 던짐. "좋은 게 뭔지 알아요?"
- **구조**: 철학 → 왜 다른지 → 우리의 답
- **말투**: 직접적이고 담백함. 과장 없음
- **특징**: 인증·성분 같은 신뢰 요소를 자연스럽게 녹임
- **길이**: 짧게 (5~7줄). 여백이 많은 게 오히려 강점

예시 구조:
```
좋은 수건의 기준, 생각해보신 적 있으세요?

부드러움? 흡수력? 가격?

저희는 거기에 하나를 더합니다.
(브랜드 철학/인증)

#해시태그
```

---

### 무신사 (패션 · 트렌드)
- **첫 줄**: "이거 모르면 손해" "이미 품절됨" 식의 긴장감
- **구조**: 훅(1줄) → 정보 나열 → 마무리 한 방
- **말투**: MZ 세대 언어. 줄임말 적당히. 솔직함
- **특징**: 계절/트렌드와 제품을 바로 연결
- **길이**: 4~8줄, 임팩트 있게

예시 구조:
```
여름 수건 고민이라면 이것만 보세요.

(핵심 포인트 3가지 나열)

나머지는 다 부가 설명입니다.

#해시태그
```

---

### Notion (생산성 · 정보형)
- **첫 줄**: 숫자로 시작. "3가지만 알면 됩니다"
- **구조**: 번호형 리스트 → 각각 한 줄 설명 → 한 줄 결론
- **말투**: 중립적이고 명확함. 군더더기 없음
- **특징**: 읽기 쉬운 구조, 저장하고 싶은 느낌
- **길이**: 중간 (7~12줄), 구조가 눈에 들어와야 함

예시 구조:
```
수건 오래 쓰는 방법, 3가지면 충분합니다.

1. (팁)
2. (팁)
3. (팁)

이것만 지켜도 확실히 달라져요.

#해시태그
```

## Threads 공통 법칙 (반응 좋은 포스트의 공통점)

### 첫 줄이 전부다
- 스크롤을 멈추게 하는 첫 줄이 없으면 아무도 안 읽음
- 질문 / 숫자 / 공감 / 반전 중 하나로 시작
- "오늘은 ~에 대해 알아볼게요" 같은 시작은 절대 금지

### 단락은 짧게
- 2~3줄 이상 이어지면 답답해 보임
- 한 줄 → 공백 → 한 줄 리듬이 Threads에 맞음
- 모바일에서 읽기 편한 호흡 유지

### 댓글 유도가 저장보다 낫다
- "어떻게 생각하세요?" "혹시 이런 경험 있으세요?" 로 마무리하면 알고리즘에 유리
- 저장 유도("이 팁 저장해두세요")보다 대화 유도가 Threads에서 더 잘 됨

### 해시태그는 3~4개가 최적
- 5개 이상이면 스팸처럼 보임
- 브랜드 태그 1개 + 주제 태그 2~3개 조합

### 이모지는 포인트로만
- 1~2개. 문장 끝에 한 번만
- 여러 개 쓰면 신뢰도가 내려감

## 테리크에 맞는 조합 추천

| 포스트 타입 | 참고 브랜드 | 핵심 |
|---|---|---|
| 감성/공감형 | 올리브영 | 오늘 날씨/계절로 시작, 친근한 말투 |
| 정보/팁형 | Notion | 번호 구조, 저장하고 싶은 실용 정보 |
| 제품/CTA형 | Lush | 철학/인증 자연스럽게, 강요 없는 권유 |
